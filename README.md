# Air Quality Monitoring system

### Backend API Routes

**Base URL**

```
     http://localhost:5050/
```

**Health Check URL**

```
     http://localhost:5050/api/health
```

**Cities URL**

```
     http://localhost:5050/api/cities
```

**Pollutants URL**

```
     http://localhost:5050/api/pollutants
```

**AQI URL**

```
     http://localhost:5050/api/measurements?city=Delhi&pollutant=AQI&limit=10
```

**KPI Data URL**

```
     http://localhost:5050/api/kpis?city=Delhi
```

**Forecast URL**

```
     http://localhost:5050/api/forecast?city=Delhi&hours=24
```

**Alerts URL**

```
     http://localhost:5050/api/alerts?city=Delhi
```
